## Customization ЁЯОи

Edit the following to customize:

### Colors
Modify `:root` variables in `styles/main.css`:
```css
--primary-green: #10B981;
--secondary-green: #D1FAE5;
/* etc. */
```

### Content
Update text in `index.html` screens to match your competition requirements.

### Logic
Modify scoring algorithm in `js/app.js` functions:
- `calculateHealthStatus()`
- `generateInsights()`
- `calculateRisks()`
